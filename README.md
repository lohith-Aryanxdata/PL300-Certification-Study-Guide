# What is PL-300?

The **Microsoft Certified: Power BI Data Analyst Associate (PL-300)** certification validates a candidate's ability to:

* Prepare data for analysis
* Model data in Power BI
* Visualize and analyze data
* Manage and secure Power BI assets
* Deploy and maintain Power BI solutions

A successful PL-300 candidate works with business stakeholders to identify requirements and deliver actionable insights through reports, dashboards, semantic models, and analytics solutions using Microsoft Power BI.

---

# Exam Skills Measured

Microsoft currently evaluates candidates across the following domains:

## 1. Prepare the Data (25–30%)

* Connect to data sources
* Clean and transform data
* Perform data profiling
* Configure data loading
* Apply Power Query transformations

## 2. Model the Data (25–30%)

* Design data models
* Create and manage relationships
* Implement star schema design
* Create calculated columns and measures
* Optimize model performance

## 3. Visualize and Analyze Data (25–30%)

* Create reports and dashboards
* Design effective visualizations
* Configure interactions
* Implement drill-through and bookmarks
* Apply advanced analytics

## 4. Manage and Secure Power BI Assets (15–20%)

* Manage workspaces
* Configure datasets
* Implement Row-Level Security (RLS)
* Monitor and troubleshoot solutions
* Apply governance and administration practices
